pageSize number <optional>
 0 represents the query page size. If pageSize==0 all the results will be returned, otherwise it will be necessary to use the nextPage method.
select string <optional>
 represents the select clause of the query
where string <optional>
 represents the where clause of the query
orderBy string <optional>
 represents the orderBy clause of the query
searchTerms string <optional>
 represents the terms used in the full text search
prefix string <optional>
 represents the definitions of the prefix used in the clausules of the query
